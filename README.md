**📍 도로명 주소 검색 서비스**
화성시재활용센터 설문 입력 편의 기능

정확한 주소 입력을 위한
간편한 도로명 주소 검색 페이지입니다.

이 페이지를 통해 사용자는 정확한 도로명 주소를 검색하고,
해당 주소를 설문에 입력할 수 있습니다.

🛠 주요 기능
도로명 주소 검색 (Kakao 주소 API 기반)
주소 자동 입력
(선택) 주소 복사 기능 지원
💻 사용 방법
"주소 검색" 버튼을 클릭합니다.
팝업에서 주소를 검색합니다.
원하는 주소를 선택합니다.
표시된 주소를 복사하여 설문(구글폼)에 입력합니다.
🔗 적용 대상
구글폼 설문조사
온라인 신청서
기타 주소 입력이 필요한 웹 기반 서비스
⚙️ 기술 스택
HTML / JavaScript
Kakao Daum 우편번호 서비스 API
📁 파일 구성
address.html : 주소 검색 기능이 구현된 메인 페이지
⚠️ 유의 사항
본 서비스는 외부 API(Kakao)를 활용합니다.
API 사용 정책 및 약관을 준수해야 합니다.
인터넷 연결이 필요합니다.
